# Go语言学习记录

这个仓库记录了我学习Go语言的整个过程和代码示例。

## 学习内容

### 基础语法
- **数组操作** (`src/base_go/arry.go`) - 一维和二维数组的定义、遍历和操作
- **位运算** (`src/base_go/bit_op.go`) - 位运算符的使用
- **常量定义** (`src/base_go/const.go`) - const关键字的使用
- **循环结构** (`src/base_go/for.go`) - for循环的各种用法
- **函数定义** (`src/base_go/func.go`) - 函数声明和调用
- **条件判断** (`src/base_go/if.go`) - if-else语句
- **跳转语句** (`src/base_go/goto.go`) - goto、break、continue的使用
- **Hello World** (`src/base_go/hello_world.go`) - Go语言入门
- **继承模拟** (`src/base_go/inherit.go`) - 通过组合实现继承效果
- **数据类型** (`src/base_go/primitive.go`) - 基本数据类型
- **结构体** (`src/base_go/struct.go`) - struct的定义和使用
- **选择结构** (`src/base_go/switch.go`) - switch语句

## 运行方式

```bash
# 运行特定文件
go run src/base_go/arry.go

# 运行整个项目
go run .
```
